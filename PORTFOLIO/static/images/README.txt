# Image Instructions

Place your images in this folder with the following filenames:

## Required Images:

1. **profile.jpg** - Your profile photo (recommended: 400x400px, square)

2. **Certificate Images:**
   - cert_1.jpg - Web Development (Elewayte)
   - cert_2.jpg - Cyber Security Add-on Certification (Tech By Heart)
   - cert_3.jpg - AI & ML Orientation (Technovalley)

3. **Project Screenshots:**
   - project_1.jpg - Automatic Timetable Generator
   - project_2.jpg - AI Chatbot

## Supported Formats:
- .jpg
- .jpeg
- .png

## Recommended Sizes:
- Profile photo: 400x400px (square)
- Certificates: 400x300px (landscape)
- Project screenshots: 500x300px (landscape)

## Notes:
- All images will have fallback placeholders if not found
- Images will be responsive and optimized for display
- Keep file sizes reasonable (under 1MB each for faster loading)
